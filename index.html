DIAGNÓSTICO 15 / 32
O Problema Real

O problema não é falta de
integração entre os sistemas.

O PROBLEMA É TRIPLO
01

Ausência de arquitetura central

Não existe um plano que defina como os sistemas se relacionam — eles só foram empilhados.

02

Ausência de ownership de dados

Nenhum sistema é, oficialmente, o dono de um determinado tipo de informação.

03

Responsabilidades indefinidas

Não há acordo claro do que cada sistema deve fazer — e do que não deve.
